## 🏃💨 Getting started

1. Clone this repo with ``git clone https://github.com/calebesg/todo-app.git``
2. Move yourself to the todo-app directory: `cd todo-app`
3. Run the ``npm install`` add all dependencies
4. Run ``npm run dev`` start app
